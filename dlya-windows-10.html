Open Broadcaster Software или OBS Studio – бесплатное кроссплатформенное приложение с открытыми исходниками для записи происходящего на дисплее, в том числе в трёхмерных играх, в файл. Также оно поддерживает трансляцию потокового видео со звуком из разных источников на популярные стриминговые платформы. Бесплатно скачать OBS Studio для Windows 10 можно по ссылке внизу.

Возможности захвата OBS Studio

OBS Studio предлагает мастер настройки для оптимизации параметров приложения под конкретные задачи: стриминг, запись в файл или комбинация двух сценариев. Окна приложения не отображаются поверх источников контента, а для простоты управления в нём реализовано управление горячими клавишами.

Программа работает с неограниченным числом источников для захвата. Ими могут быть игра, окно приложения, воспроизводимое в плеере видео, картинки, слайд-шоу, тексты, различные устройства захвата (IP-, web-камеры). Звук можно записывать из любой программы, выхода звуковой платы, микрофона и прочих внешних устройтв. Если планируете дальнейший монтаж видео, аудио из каждого источника можно записывать в отдельную звуковую дорожку.

Из возможностей обработки OBS поддерживает фильтры. Для аудио это сжатие, шумоподавление, усиление, трёхполосный эквалайзер и другие. Для добавления сторонних эффектов реализована поддержка VST-плагинов. Что касается картинки, установить сторонние видеофильтры нельзя, однако встроенных хватит для:

  • цветокоррекции: управления цветом, яркостью, контрастностью;
  • масштабирования картинки с разными фильтрами;
  • изменения соотношения сторон изображения;
  • прокрутки в любом направлении;
  • замены фона (хромакей);
  • кадрирования – обрезки изображения.

Профили, трансляция, субтитры

Для быстрого переключения между наборами параметров OBS Studio поддерживает профили с настройками, их импорт и экспорт из файла, а также коллекции источников для быстрого формирования сцен. Приложение умеет транслировать потоки на 60+ стриминговых сервисов после авторизации в аккаунте и перепаковывать записанные видео. В последних версиях приложения появилась экспериментальная функция наложения субтитров.

Особенности OBS Studio:

  • Размещение источников на разных слоях.
  • Модуль хромакея.
  • Фильтры для обработки звуковых и видеодорожек.
  • Гибкие настройки звука и видео.
  • Совместимость с десятками сервисов для трансляций.
  • Встроенные аудио- и видеофильтры.

OBS Studio – бесплатный инструмент для записи экранного видео, захвата видеопотоков и звука из внешних устройств для сохранения в файл или трансляции в сеть. 

Как установить?

Скачайте последнюю версию OBS Studio для виндовс 7 по ссылке, запустите полученный файл, кликните «Next» в обоих окнах.

Укажите каталог для разворачивания приложения, кликните «Install».

Нажмите "далее", затем программа начнет установку и можно ей пользоваться.
