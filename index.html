OBS для записи экрана ПК

OBS Studio – сборник программ для записи, обработки, сохранения, кодирования и трансляции потокового видео со звуком. Приложение поддерживается десятками добровольцев со всего мира, в число спонсоров входят крупные IT-корпорации: AMD, Nvidia, Meta, Google. Оно умеет захватывать видео с различных источников, включая видеоустройства, например, веб- и IP-камеры. По сети контент передаёт через протокол RTMP, есть профили для стрима на YouTube, Twitch.

Среди особенностей программы можно выделить:

  • Интеграция с популярными потоковыми сервисами.
  • Функция захвата видео и звука с разных источников, тонкая настройка потоков.
  • Инструменты обработки видео- и аудиодорожек.
  • Режим студии или предварительного просмотра.
  • Управление комбинациями клавиш.
  • Создание профилей для работы в различных режимах.

Сцены, источники

OBS Studio захватывает аудио- и видео потоки с ограниченного производительностью вашего компьютера / ноутбука количества сцен: весь экран, веб-камера, окна, фреймы, приложения, меню и прочие. Между источниками можно плавно переходить благодаря видеопереходам, захватывать видеоряд из нескольких мониторов. В качестве источников видео могут выступать:

  • окно программы, её фрейм или иной элемент интерфейса;
  • трёхмерная или двухмерная игра, в том числе запущенная в полноэкранном режиме;
  • картинка, фоновый цвет;
  • видеопроигрыватель;
  • слайд-шоу, презентация;
  • устройство захвата видеоряда (различные камеры, рекордеры);

Также программа захватывает аудио из любой программы, игры или видеовыхода.

Каждый источник настраиваемый, причём перечень параметров зависит от его типа. Для слайд-шоу можете выбрать слайды – изображения, задействовать аппаратное декодирование для ускорения сжатия видеоряда.

Поддержка слоёв позволяет упорядочивать изображения – располагать их одно над другим. Используя OBS можно обойтись без видеоредакторов. К вашим услугам 12 настраиваемых видеофильтров: работа с масками, цветокоррекция, хромакей, управление резкостью, размытие, изменение соотношения сторон. Последовательность фильтров, как и слоёв видео в сцене, можно менять. В режиме студии можно сравнить результат обработки: просмотреть исходник и отредактированный видеоряд.

Звук, декодирование

Микшер позволяет менять уровень громкости аудио из разных источников, добавлять на аудиопоток сколько угодно из десятка доступных аудиофильтров, настраивать их. Например, через OBS Studio в пару кликов можно снизить уровень шума, синхронизировать звук, усилить его. Есть поддержка VST-плагинов для продвинутой обработки аудио. Функция записи звука и каждого источника в отдельную аудиодорожку облегчит дальнейшую обработку звука в видеоредакторах.

Приложение может использовать кодек H.264 для кодирования видеопотоков. В нём есть два режима вывода информации: простой и расширенный. Если в первом доступны настройки трансляции и записи, во втором добавляются параметры буфера и звука. Используя OBS Studio, сможете контролировать каждый аспект качества сохраняемого либо потокового видео: битрейт, разрешение, кодировщики видео и звука, формат файла, количество звуковых дорожек.

Настройки

У OBS Studio богатый набор параметров конфигурации. Она позволяет привязывать источники к краям экрана или друг к другу, центру экрана, вертикали или горизонтали. Для удобства эксплуатации разработчик позаботился о настройке поведения приложения: вы можете автоматизировать десяток операций при определённых условиях, например, включить захват видео на диск в момент запуска трансляции или прятать окна подтверждения действий.

OBS организует стрим на одну или несколько из десятка поддерживаемых площадок после ввода данных для авторизации. При этом поток можно защитить при помощи ключа – сторонние пользователи не подключатся к вашему вещанию.

Скачайте последнюю версию OBS Studio по ссылке, запустите полученный файл, кликните «Next» в обоих окнах.
